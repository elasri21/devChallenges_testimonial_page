# Testimonial Page

## Description

This project is a simple testimonial page that showcases reviews and feedback. It includes a section for general reviews and a section for company testimonials. The page uses Font Awesome for star icons and custom icons for checklists.

## Features

- **Review Section:** Displays general information about review guidelines and checklists.
- **Company Testimonials:** Shows testimonials from various companies with star ratings.

## Technologies Used

- **HTML:** For the page structure and content.
- **CSS:** (Assumed) For styling the page. You need to include a `style.css` file to style the page.
- **Font Awesome:** For star icons.
- **Custom Icons:** SVG icons for checklists and company logos.

## Installation

1. **Clone the Repository:**

   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd <project-directory>
   ```

3. **Open the HTML File:**

   Open `index.html` in your web browser to view the page.

## Usage

1. **HTML Structure:**

   - **Review Section:** The `reviews` div contains a header and a checklist.
   - **Company Testimonials:** The `companies` div contains testimonials with star ratings.

2. **CSS Styling:**

   - Ensure you have a `style.css` file to style the HTML elements. This file should be located in the same directory as the `index.html` file.

3. **Assets:**
   - The page references SVG icons located in the `./assets/` directory. Ensure these assets are present in your project directory.
